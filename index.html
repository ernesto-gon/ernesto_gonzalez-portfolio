Sobre Mí

¡Hola! Soy Ernesto, un apasionado estudiante de informática Segoviano que está viviendo en Valencia. Actualmente estoy cursando la FP superior de Desarrollo de Aplicaciones Web, donde estoy adquiriendo una base sólida en el desarrollo de software y aplicaciones web. Desde siempre me ha fascinado cómo la tecnología puede resolver problemas reales y mejorar la vida de las personas. Mis principales intereses se centran en la programación y el desarrollo front-end, el diseño de interfaces de usuario (UI/UX) y la creación de experiencias web accesibles e intuitivas para todos. Mi objetivo profesional a corto plazo es conseguir experiencia a través de prácticas y una base sólida para continuar mi formación académica, además de poder llegar a ser un experto en el sector. A largo plazo, aspiro a convertirme en un desarrollador full-stack especializado en tecnologías modernas.

Proyectos

Programación en Java de un programa que adivina un número que está pensando el usuario entre 1 y 100

En este proyecto, he tenido que realizar en Java un programa donde se tiene que adivinar un número entre 1 y 100, el cuál está pensando el usuario. Este tiene que indicar si es mayor, menor o igual al número que aparece en pantalla. Este proyecto lo he realizado en la asignatura de programación.

Tecnologías utilizadas:

  • Java
  • Bucles en Java

Tecnologías utilizadas:
